1.  Download the latest version(3.4 and later) of Python from https://www.python.org/downloads/
2.  Run the installer
    Important: Check the box that says "Add Python to PATH" before clicking "Install Now".
3.  Open the Snake_Master folder using the command prompt
4.  Type "pip install pygame"
5.  After installing pygame, run the snake_master(.exe file) to enjoy
